# Mentark Path Finder - Premium PDF Report Schema

## Document Structure

### Cover Page
- **Dark background** with gold/cyan accents
- **Mentark branding** (logo + title)
- **Student information box** (name, stream, generation date)
- **Confidential footer**

### Table of Contents (if >5 sections)
- Auto-generated based on available sections
- Page numbers for easy navigation

### 1. Executive Summary
- Overview of the complete report
- Purpose and value proposition

### 2. Your Core Strengths
- List of identified strengths
- Visual bullet points with gold accents

### 3. Your Aptitude DNA (if available)
- **5 dimensions**: Logic, Creativity, Social, Resilience, Financial
- **Progress bars** with percentage scores
- Visual representation of core strengths

### 4. Stream Fit Analysis (if available)
- **Comparison chart** of all streams
- **Score bars** with color coding:
  - Gold: Recommended stream
  - Green: High fit (>70%)
  - Yellow: Medium fit (40-70%)
  - Gray: Low fit (<40%)
- Visual indicators for best match

### 5. Recommended Career Paths
- **Card-based layout** for each path
- Path name with bold styling
- "Why this path" descriptions
- Numbered for easy reference

### 6. Expanded Career Opportunities (if available)
- **Three categories**:
  - **Niche & Emerging Careers**: Quantum Computing, Robotics, Space Systems, Blockchain, Climate Tech
  - **International Opportunities**: Silicon Valley, Europe, Middle East, Singapore
  - **Government Careers**: ISRO, DRDO, BHEL, BARC, IAS/IFS
- Each opportunity includes:
  - Title with growth potential badge
  - Description
  - Salary range (India + International)
  - Requirements list
  - "Why consider" insight

### 7. Personality Insights (if available)
- **Personality type** (e.g., "Analytical Innovator")
- **Description** of personality profile
- **Strengths** list with green indicators
- **Growth areas** with yellow indicators

### 8. Your Learning Style (if available)
- Primary learning style
- Secondary learning style
- Description and recommendations

### 9. Subject Recommendations (if available)
- Subject names
- Reasons for recommendation
- Personalized explanations

### 10. Your 2-Year Roadmap
- **Title and description**
- **Milestones** (detailed):
  - Milestone number and month range
  - Title and description
  - **Actions** (checklist format)
  - **Resources** (NEW - Detailed):
    - Resource type badges (VIDEO, ARTICLE, COURSE, BOOK, TOOL)
    - Resource titles
    - Descriptions explaining value
    - Color-coded by type
- **Monthly Action Plan**:
  - Year 1 breakdown (months 0-3, 4-6, 7-9, 10-12)
  - Year 2 breakdown (months 13-15, 16-18, 19-21, 22-24)
  - Specific focus areas with actions

### 11. Success Tips & Best Practices
- Personalized tips based on strengths
- Actionable advice
- Gold bullet points

### Footer (All Pages)
- Page numbers (X of Y)
- "Generated by Mentark Path Finder"
- Subtle border line

## Design Specifications

### Color Scheme
- **Primary**: Gold (#FFD700) - Headers, accents, highlights
- **Secondary**: Cyan (#00E6FF) - Secondary elements
- **Accent**: Pink (#FF006E) - Special highlights
- **Success**: Green (#22C55E) - Positive indicators
- **Warning**: Yellow (#EAB308) - Attention items
- **Text**: Dark slate (#1E293B) - Main content
- **Borders**: Light slate (#94A3B8) - Subtle dividers

### Typography
- **Headers**: Helvetica Bold, 14-16pt
- **Subheaders**: Helvetica Bold, 11-12pt
- **Body**: Helvetica Normal, 9-10pt
- **Small text**: Helvetica Normal, 7-8pt

### Layout Elements
- **Margins**: 15mm all sides
- **Section spacing**: 5-10mm between sections
- **Bordered boxes**: 2mm rounded corners
- **Progress bars**: 4-5mm height
- **Badges**: Color-coded with rounded corners

### Visual Elements
- **Section headers**: Gold background bars with white text
- **Cards**: Light gray backgrounds with borders
- **Progress indicators**: Color-coded bars
- **Bullet points**: Colored circles matching category
- **Resource badges**: Type-specific colors

## Data Requirements

The PDF generator accepts the following data structure:

```typescript
interface ReportData {
  studentName?: string;
  stream: string;
  strengths: string[];
  paths: Array<{ name: string; why: string }>;
  roadmap: {
    title: string;
    description: string;
    milestones: Array<{
      month_range: string;
      title: string;
      description: string;
      actions: string[];
      resources: Array<{
        type: 'video' | 'article' | 'course' | 'book' | 'tool';
        title: string;
        url?: string;
        description: string;
      }>;
      skills_to_build?: string[];
      exam_prep?: string;
      career_connection?: string;
    }>;
    monthly_plan: {
      year_1: {
        months_0_3: string[];
        months_4_6: string[];
        months_7_9: string[];
        months_10_12: string[];
      };
      year_2: {
        months_13_15: string[];
        months_16_18: string[];
        months_19_21: string[];
        months_22_24: string[];
      };
    };
    success_tips: string[];
    career_exposure?: string[];
    exam_timeline?: string[];
    budget_friendly_options?: string[];
  };
  language: 'en' | 'hi' | 'mr';
  
  // Optional comprehensive data
  result?: QuizResult;
  aptitudeDNA?: Array<{ axis: string; value: number }>;
  streamFit?: Array<{ stream: string; score: number }>;
  expandedOpportunities?: ExpandedCareerOpportunity[];
  traitScores?: any;
  personalityInsights?: any;
  completeTraitProfile?: any[];
  learningStyle?: any;
  whoYouAreNow?: any;
  subjectRecommendations?: any[];
}
```
